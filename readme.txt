# Jensen & Juhl Website

## Project Overview
This repository contains the source code for the official website of Jensen & Juhl, a production group based in Los Angeles. The website serves as a hub for showcasing Jensen & Juhl's brand identity, services, and projects across event design, event fabric manufacturing, and apparel branding.

## Features
- **Responsive Design:** Optimized for various devices and screen sizes.
- **HTML5 and CSS3:** Modern web standards for clean and semantic code.
- **Custom GSAP Animations:** Interactive preloader and dynamic content transitions.
- **SEO-Friendly:** Structured meta tags and descriptions to improve visibility.
- **Fast Loading:** Lightweight assets and efficient use of resources.

## Technologies Used
- **HTML5**: Markup structure.
- **CSS3**: Styling and responsive design.
- **JavaScript**: Client-side interactivity.
- **Python**: Local server for development.
- **GSAP**: Animations for a smooth user experience.

## Installation
To run the website locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/jensenandjuhl-site.git
   ```

2. Navigate to the project directory:
   ```bash
   cd jensenandjuhl-site
   ```

3. Start a local Python server:
   ```bash
   python3 -m http.server
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```
